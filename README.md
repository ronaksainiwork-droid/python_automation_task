# Python Automation

This repository contains Python automation scripts related to Cloud Computing and Data Engineering.

## Projects

### 1. AWS EC2 Instance Creation
- Python + Boto3
- Automated EC2 Instance Launch
   ## How to Run
   
   1. Install Boto3
      pip install boto3
   
   2. Configure AWS Credentials
   
   3. Run the script
      python ec2_instance_creation.py
   
   ## Output
   EC2 Instance Created Successfully

  
### 2. Kaggle to AWS S3 Data Pipeline
- Download Dataset using Kaggle API
- Create AWS S3 Bucket
- Upload Dataset to S3 using Boto3

## Technologies Used
- Python
- AWS
- Boto3
- Kaggle API
