# AWS EC2 Instance Automation

This project demonstrates how to create an AWS EC2 instance using Python and Boto3.

## Technologies Used
- Python
- AWS EC2
- Boto3

## Features
- Programmatically create EC2 instances
- Basic AWS cloud automation
- Uses Boto3 SDK

## How to Run

1. Install Boto3
   pip install boto3

2. Configure AWS Credentials

3. Run the script
   python ec2_instance_creation.py

## Output

EC2 Instance Created Successfully
